Thread structure

Each exchange is 3 items: Marcus's question → your answer (right-aligned green bubble) → Marcus's reaction (stays permanently, italic, left-indented).

Active question

Current question gets a green border instead of grey — visually marks "this is where you are." Input + arrow button sit inline below it, indented to align with the bubble.

Progress dots

6 dots below the input. Terracotta = done, dark green = active, grey = upcoming.

Editing previous answers

The editable chips are removed — they don't fit the thread metaphor. If someone needs to correct an answer, they'd just restart (the thread is short, ~2 min).

Scroll behavior

Thread grows downward. After each answer + reaction, auto-scroll keeps the active input in view. No jumping.
